# 📊 Live Dashboard

Real-time monitoring interface for the API Yellow Pages autonomous system.

## Features

- Live system status
- API health metrics
- Category browser
- Activity log
- Performance analytics
- Auto-refresh every 30 seconds

## Usage

Open `index.html` in a web browser or serve via HTTP:

```bash
cd dashboard
python3 -m http.server 8000
```

Then visit: http://localhost:8000

## Metrics Displayed

- Total APIs (verified, caution, failed)
- Truth scores
- Response times
- Uptime percentages
- Discovery status
- Recent activity
